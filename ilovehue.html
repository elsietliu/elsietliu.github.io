I Love Hue

Creating a game using OCaml

Work in: Backend development, OCaml
Done for: Data Structures and Functional Programming, Cornell University
Team: Myself and two other students
Timeline: September 2021 - December 2021

The Project

We created this game as the final project of our Data Structures and Functional Programming Course. The user recieves a pre-scrambled map of different colored blocks and must swap them around to form a gradient.

Key Features

  • Level select
  • Map Generation
  • Move tiles by clicking two tiles in succession to swap
  • Reset level
  • Return to main menu
  • Record number of swaps
  • Display score at end of level

Gameplay

The gameplay is relatively simple. There are nine levels of increasing difficulty for the user to choose from. To play the game, the user swaps blocks by clicking on two different blocks consecutively, causing them to swap places.

Once the user has successfully completed the level, the end screen displays the total number of swaps used for that playthrough, as well as the best score the user has gotten if they had played previously. From the end screen, the user also has the ability to replay or quit the game.
