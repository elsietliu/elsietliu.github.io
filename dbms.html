Database Management System

Building a functional DBMS

Work in: Backend development, Java, PostgreSQL
Done for: Databases Practicum, Cornell University
Team: Myself and two other students
Timeline: August 2022 - December 2022

The Project

This project was done for my Databases Practicum course at Cornell University. The goal of the project was to develop a simple database management system with the following features:

  • Processes SELECT-FROM-WHERE queries
  • Implements different operator versions
  • Optimizes plans via query optimization

The project was broken down into four main phases:

  1. SQL interpreter on small data, in-memory evaluation
  2. Scaling to bigger files
  3. Data indexing
  4. Query optimization

The Team

I worked with a team of two other students in the class. All of us were also taking the corresponding course, Introduction to Database Systems, so we were impolementing the concepts as we were learning them.
